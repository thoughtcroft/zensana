# Zensana

This gem will take an Asana project export JSON file and import it into
ZenDesk using the API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'zensana'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install zensana
